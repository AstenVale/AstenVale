Help

How the admin panel works

Releasing a Case

1. Go to Case Files and find the case you want to release.
2. Toggle the Released switch to ON.
3. Optionally set a Release Date for the countdown timer.
4. Click Publish Changes in the top bar.
5. GitHub Pages rebuilds in ~30 seconds. The case is now live.

Adding a New Season

Click + Add Season in the Case Files tab.
Fill in the season title and add cases one by one.
All new cases default to unreleased until you toggle them on.

Publish Changes

Every save commits directly to your GitHub repo.
The site updates automatically within ~30 seconds.
All changes are recorded in your GitHub commit history.
